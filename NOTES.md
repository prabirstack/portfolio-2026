# Learning Notes

## Dependencies

| Package                     | Description                                                                             |
| --------------------------- | --------------------------------------------------------------------------------------- |
| next-themes                 | Theme management for Next.js apps with automatic dark/light mode switching              |
| react-hook-form             | Form state management (values, errors, submission) via React hooks                      |
| zod                         | Runtime schema validation for forms, APIs, environment variables, and other data        |
| uploadthing                 | Simple file upload handling for full-stack TypeScript apps                              |
| resend                      | API service for sending transactional emails from your application                      |
| postgres                    | Node.js driver that connects your app to PostgreSQL                                     |
| @t3-oss/env-nextjs          | Type-safe environment variable validation for Next.js apps                              |
| @trpc/react-query           | Integrates tRPC with React Query for typed API hooks with caching and loading states    |
| @tanstack/react-query       | Server state management library for caching, fetching, and syncing API data             |
| @hookform/resolvers         | Connects react-hook-form with validation libraries like Zod                             |
| drizzle-orm / drizzle-kit   | Type-safe ORM for writing SQL queries in TypeScript                                     |
| better-auth                 | Authentication library for managing user login, sessions, and security                  |
| motion                      | Animation library for building smooth UI interactions in React                          |
| @trpc/client / @trpc/server | End-to-end typesafe API calls between client and server without REST or code generation |
| radix-ui                    | Accessible unstyled UI primitives like dialogs, tabs, and dropdowns                     |
| class-variance-authority    | Utility for managing dynamic Tailwind class variants in UI components                   |
| tailwind-merge / clsx       | Utilities for combining and resolving Tailwind CSS class names                          |
